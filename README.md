# 🔳 Gerador de QR Code

Interface web moderna pra gerar QR Codes de Wi-Fi, links, telefones e endereços em segundos. 100% client-side, sem backend.

![preview](https://via.placeholder.com/800x400/0f172a/38bdf8?text=Preview+do+Gerador+de+QR+Code)

## ✨ Funcionalidades

- **4 tipos de QR Code**: Wi-Fi, Link, Telefone e Endereço
- **Download em PNG**: Baixa a imagem em alta qualidade
- **Exportar PDF**: Gera PDF pronto pra impressão
- **Layout responsivo**: Funciona no celular e desktop
- **Zero dependência**: HTML + JS puro, roda direto no navegador
- **Dark mode**: Design moderno com glassmorphism
